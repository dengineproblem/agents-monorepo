### 3. Полный рабочий процесс (flow)

#### Сценарий 1: Генерация с референсом

```
1. Заполнение текстов
   ┌─────────────┐
   │ Offer       │ ← Вводите или генерируете AI
   │ Bullets     │
   │ Profits     │
   │ CTA         │
   └─────────────┘
          ↓
2. Загрузка референса
   ┌─────────────┐
   │ 📸 Upload   │ ← Загружаете изображение
   │ [Preview]   │
   └─────────────┘
          ↓
3. Генерация
   ┌─────────────┐
   │ 🎨 Generate │ ← Нажимаете кнопку
   └─────────────┘
          ↓
4. Результат
   ┌─────────────┐
   │ [Creative]  │ ← Получаете креатив
   │ [Edit btn]  │
   └─────────────┘
```

#### Сценарий 2: Редактирование

```
1. Есть сгенерированный креатив
   ┌─────────────┐
   │ [Creative]  │
   │ [Edit btn]  │ ← Нажимаете "Редактировать"
   └─────────────┘
          ↓
2. Режим редактирования
   ┌─────────────┐
   │ [Creative]  │
   │ [Prompt]    │ ← Вводите инструкции
   │ [Apply btn] │
   └─────────────┘
          ↓
3. Применение
   ┌─────────────┐
   │ Loading...  │ ← Генерация с текущим как референсом
   └─────────────┘
          ↓
4. Обновленный креатив
   ┌─────────────┐
   │ [New ver.]  │ ← Получаете улучшенную версию
   │ [Edit btn]  │ ← Можете редактировать снова
   └─────────────┘
```

## 🎨 Визуальные элементы

### Иконки
- 📸 Upload - для загрузки референса
- 🖊️ Edit - для редактирования креатива
- ✨ Sparkles - для применения изменений
- ✕ X - для удаления референса
- ✓ Checkmark - подтверждение загрузки

### Цветовая схема
- **Синяя панель** - режим редактирования
- **Зеленая панель** - подтверждение загрузки
- **Желтая панель** - советы и подсказки

### Состояния кнопок
- **Активная** - "Применить изменения"
- **Загрузка** - "Применяю изменения..." (с иконкой спиннера)
- **Отключена** - когда нет текста в промпте

## 💡 Подсказки в интерфейсе

1. **При загрузке референса:**
   ```
   💡 Совет: Используйте референс для брендинга,
   стилизации или композиции
   ```

2. **В режиме редактирования:**
   ```
   💡 Опишите, что нужно изменить. Текущее изображение
   будет использовано как референс.
   ```

3. **При превышении лимита:**
   ```
   ⚠️ У вас закончились генерации креативов.
   Приобретите дополнительный пакет.
   ```

## 🔄 Интерактивность

### Drag & Drop (будущее улучшение)
Возможность перетаскивания изображений в область загрузки.

### Предпросмотр
Автоматический предпросмотр референсного изображения после загрузки.

### Валидация
- Проверка типа файла (только изображения)
- Проверка размера (до 10MB)
- Проверка заполнения промпта редактирования
