# Креативы

Папка для хранения креативов по аккаунтам.

## Структура

```
creatives/
├── README.md           ← Этот файл
├── {account_name}/     ← Папка для каждого аккаунта
│   ├── index.md        ← Реестр креативов
│   ├── video_1.mp4     ← Файлы креативов
│   ├── image_1.jpg
│   └── ...
```

## Формат index.md

```markdown
# Креативы: {Account Name}

| creative_name | Файл | Формат | Направление | Описание |
|---------------|------|--------|-------------|----------|
| implant_v1 | implant_v1.mp4 | video | Имплантация | Первое видео |
| veneers_carousel | veneers_1.jpg | image | Виниры | Карусель |
```

## Как работает

1. Пользователь добавляет креативы в папку аккаунта
2. Регистрирует их в `index.md` с `creative_name` и направлением
3. При запуске `/ads-optimizer`:
   - Скилл читает `index.md`
   - Сравнивает с активными объявлениями
   - Находит неиспользуемые креативы
   - Предлагает запустить их при наличии бюджета

## Важно

- `creative_name` должен быть уникальным в рамках аккаунта
- `creative_name` используется в названиях AdSet и Ad
- Формат: video, image, carousel
- Направление должно совпадать с Campaign в аккаунте
