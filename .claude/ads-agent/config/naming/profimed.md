# Naming Convention: Profimed

Account ID: act_1106872004544227

---

## Формат

```
{creative_tag}_{аудитория}_{гео}
```

| Часть | Описание | Примеры |
|-------|----------|---------|
| `creative_tag` | Название креатива из реестра | `полный чекап бесплатно`, `восстановим 7000` |
| `аудитория` | Тип аудитории или возраст | `25-45`, `lookalike`, `ретаргет` |
| `гео` | Город или регион | `алматы`, `казахстан` |

---

Чекап_25-45_алматы

---

## Типы аудиторий

| Значение | Описание |
|----------|----------|
| `25-45`, `35-55` и т.д. | Возрастной диапазон |
| `lookalike` | Похожая аудитория |
| `ретаргет` | Ретаргетинг (посетители, взаимодействия) |
| `broad` | Широкая аудитория без интересов |
| `интересы` | Таргетинг по интересам |

---

## Гео

| Значение | Описание |
|----------|----------|
| `алматы` | Только Алматы |
| `казахстан` | Вся страна |
| `астана` | Только Астана |

## Парсинг

```python
def parse_ad_name(ad_name):
    """Парсит имя объявления на компоненты"""
    parts = ad_name.rsplit("_", 2)
    if len(parts) == 3:
        return {
            "creative_tag": parts[0],
            "audience": parts[1],
            "geo": parts[2]
        }
    return {"creative_tag": ad_name, "audience": None, "geo": None}
```

```
Ad Name: "полный чекап бесплатно_25-45_алматы"
→ creative_tag: "полный чекап бесплатно"
→ audience: "25-45"
→ geo: "алматы"
```
