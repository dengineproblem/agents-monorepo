# Naming Convention для объявлений

Система именования позволяет группировать объявления по креативам и анализировать их эффективность.

---

## Формат имени Ad

```
{creative_tag}_{variant}_{audience}
```

| Часть | Описание | Примеры |
|-------|----------|---------|
| `creative_tag` | Уникальный идентификатор креатива | `kitchen`, `bathroom_promo`, `newyear` |
| `variant` | Версия креатива (опционально) | `v1`, `v2`, `short`, `long` |
| `audience` | Аудитория/таргетинг (опционально) | `30-45`, `lookalike`, `retarget` |

---

## Примеры

### Простое именование
```
kitchen_v1
kitchen_v2
bathroom_v1
```

### С аудиторией
```
kitchen_v1_30-45_msk
kitchen_v1_lookalike
kitchen_v2_retarget
```

### Для A/B тестов
```
promo_hook1_broad
promo_hook2_broad
promo_cta1_broad
promo_cta2_broad
```

---

## Правила

### Обязательно
- `creative_tag` — ВСЕГДА присутствует
- Только латиница и цифры
- Разделитель: `_` (нижнее подчёркивание)
- Без пробелов

### Рекомендуется
- Короткие теги (до 15 символов)
- Понятные названия на английском
- Версионирование: `v1`, `v2`, `v3`

### Избегать
- Кириллицы в тегах
- Дат в имени (используй версии)
- Слишком длинных имён (>50 символов)

---

## Как парсить

```
Ad Name: "kitchen_v1_30-45_msk"

Разбиение по "_":
- parts[0] = "kitchen"     → creative_tag
- parts[1] = "v1"          → variant (если начинается с v + цифра)
- parts[2+] = "30-45_msk"  → audience_info
```

**Группировка:**
- По креативу: `kitchen_*` — все объявления с этим видео
- По варианту: `kitchen_v1_*` — конкретная версия
- Полное совпадение: точный creative_tag + variant

## Использование в Skills

### /creative-analyzer
Автоматически группирует ads по `creative_tag` и показывает:
- Агрегированные метрики по креативу
- Сравнение вариантов (v1 vs v2)
- Risk Score по креативу

### /campaign-manager
При создании ad предлагает:
- Выбрать существующий `creative_tag` из реестра
- Или создать новый и добавить в `config/creatives.md`

### /ads-reporter
Breakdown по креативам в отчётах.

---

## Миграция существующих ads

Если объявления уже созданы без convention:
1. Переименовать через Facebook Ads Manager
2. Или использовать `update_ad` для изменения имени
3. Добавить теги в `config/creatives.md`
